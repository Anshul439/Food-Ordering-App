# 🍔 Food Ordering App

A simple food ordering app built with **React.js** and styled using **Tailwind CSS**.  
Deployed on **Vercel**.

## 🚀 Features
- Browse food items  
- Add items to cart  
- Contact & About pages  
- Styled with Tailwind CSS  
- Unit tests with Jest  

## 🛠 Tech Stack
- React.js  
- Redux
- Tailwind CSS  
- Parcel Bundler  
- Jest (for testing)

## 📦 Installation
```bash
git clone https://github.com/Anshul439/food-ordering-app.git
cd food-ordering-app
npm install
npm start
